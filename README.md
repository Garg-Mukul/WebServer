Multi-Threaded Java socket server
A robust implementation of a concurrent Client-Server architecture using Java Sockets.
This project demonstrates how to handle multiple simultaneous client connections using Java's threading model and functional interfaces.
Features:
1) Multi-Threaded server
2) Functional design
3) Resource management

Architecture:
1) The server
2) The client

How to run?
Prerequisites: JDK, A terminal or IDE
1) Compile the files
2) Start the server first
3) Run the client(in a separate terminal)
